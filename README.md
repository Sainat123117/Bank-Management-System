# Bank-Management-System
A robust, console-based Bank Management System written in C. Features include account creation, money transfer, deposits, withdrawals, and transaction history, utilizing file handling for data persistence and a Makefile for build automation.

A low-level implementation of a banking system developed in C on a Linux environment. This project focuses on **systems programming concepts** including file stream manipulation, modular compilation with Makefiles, and efficient memory usage.
